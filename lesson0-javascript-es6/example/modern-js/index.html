Essential JavaScript concepts for Vue 3 development

1. Arrow Functions

Arrow functions are crucial in Vue 3 for event handlers and composables.

// Traditional function function traditionalGreet(name) { return `Hello, ${name}!`; } // Arrow function (preferred in Vue 3) const arrowGreet = (name) => { return `Hello, ${name}!`; }; // Concise arrow function const conciseGreet = name => `Hello, ${name}!`; // Arrow function with multiple parameters const calculateArea = (width, height) => width * height;

2. Destructuring

Essential for working with Vue props and reactive state.

// Object destructuring const student = { name: "Alice", age: 21, major: "Computer Science", gpa: 3.8 }; const { name, age, major } = student; console.log(name); // 'Alice' // Array destructuring const scores = [85, 92, 78, 96]; const [first, second, ...rest] = scores; // Function parameter destructuring (common in Vue composables) const displayStudent = ({ name, age, gpa }) => { return `${name} is ${age} years old with GPA ${gpa}`; };

3. Template Literals

For dynamic strings in Vue templates and composables.

const studentName = "Sarah"; const grade = "A"; const course = "Vue 3 Development"; // Old way const oldMessage = "Student " + studentName + " received grade " + grade + " in " + course; // Template literal (preferred) const newMessage = `Student ${studentName} received grade ${grade} in ${course}`; // Multi-line strings const htmlTemplate = `

Welcome ${studentName}

Your grade: ${grade}

Course: ${course}

`;

4. Spread Operator

For copying and merging objects/arrays in Vue reactive state.

// Array spreading const fruits = ["apple", "banana"]; const moreFruits = ["orange", "grape"]; const allFruits = [...fruits, ...moreFruits]; // Object spreading (very important for Vue reactive state) const student = { name: "Alice", age: 21 }; const updatedStudent = { ...student, grade: "A", major: "CS" }; // Function arguments const numbers = [1, 2, 3, 4, 5]; const max = Math.max(...numbers);
